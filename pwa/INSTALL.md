# AutoBot Phone Launcher

Public launcher page:

https://ha89ter-netizen.github.io/v2/pwa/

This page lets a user add AutoBot to the phone home screen like a small app.
When the icon is tapped, it opens the AutoBot Telegram chat directly.

## iPhone

1. Open `https://ha89ter-netizen.github.io/v2/pwa/` in Safari.
2. Tap the Share button.
3. Choose `Add to Home Screen`.
4. Name it `AutoBot`.
5. Tap `Add`.
6. Open AutoBot from the new icon on the home screen.

If iPhone asks whether it can open Telegram, tap `Open`.

## Android

1. Open `https://ha89ter-netizen.github.io/v2/pwa/` in Chrome.
2. Tap the menu button.
3. Choose `Install app` or `Add to Home screen`.
4. Confirm the installation.
5. Open AutoBot from the new icon on the home screen.

## Important

- Telegram must be installed on the phone.
- If Telegram does not open automatically, tap `Open in Telegram`.
- If the browser opens first, this is normal on some phones. The launcher will still take the user to Telegram.
- This shortcut does not replace the Telegram bot. It is a faster entry point for users.

## Russian quick guide

Ссылка для пользователей:

https://ha89ter-netizen.github.io/v2/pwa/

iPhone:

1. Открыть ссылку в Safari.
2. Нажать `Поделиться`.
3. Нажать `На экран Домой`.
4. Назвать `AutoBot`.
5. Нажать `Добавить`.

Android:

1. Открыть ссылку в Chrome.
2. Нажать меню.
3. Выбрать `Установить приложение` или `Добавить на главный экран`.
4. Подтвердить.

После этого у пользователя появится иконка AutoBot на телефоне. Нажатие на нее
будет открывать чат с ботом в Telegram.

## Product onboarding recommendation

Make this a recommended onboarding step:

1. User starts the Telegram bot.
2. Bot solves the first useful scenario: diagnosis, service search or parts search.
3. Bot sends: `Want faster access next time? Add AutoBot to your phone home screen.`
4. Bot gives the launcher link: `https://ha89ter-netizen.github.io/v2/pwa/`.

This makes AutoBot feel closer to an app while keeping the simplicity of Telegram.
